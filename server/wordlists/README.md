# Wordlists for subdomain and directory bruteforce

Subdomain and directory wordlists are derived from [SecLists](https://github.com/danielmiessler/SecLists) (Discovery/DNS and Discovery/Web-Content) for use in EASM subdomain bruteforce and OSINT directory bruteforce.

- **subdomains.txt**: Subdomain prefixes (subset of SecLists `subdomains-top1million-5000.txt`).
- **directories.txt**: Web paths (subset of SecLists `common.txt`), normalized with leading `/`.
